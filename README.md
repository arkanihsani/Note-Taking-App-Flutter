Prerequisite
-Flutter
-Firebase Project
Cara Inisialisasi App
-Buat project dalam firebase dan aktifkan firestore database
-Login di firebase CLI dengan "firebase login"
-Buat project flutter dengan "flutter create <folder name>"
-Dalam root project flutter, run "dart pub global activate flutterfire_cli" dan "flutterfire configure" di terminal
-jalankan "flutter run" dalam terminal
